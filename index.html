結論：掛單是實現 RR>2 最自然的方式。在支撐等待買進，停損在支撐下方很近，目標在前高很遠，RR 由結構決定，不需要主觀調整。
方法一：支撐 / 阻力回測掛單（最常用，RR 最高）
1
找支撐/阻力區（1小時圖）：選近 10–20 根 K 棒，找價格曾被「觸及 + 反彈」≥ 2 次的價格區間（寬度通常 5–15 pips）。越多次觸及越有效。
2
確認大趨勢方向：看 1小時 EMA20 斜率。EMA20 向上 → 只做多（掛 Buy Limit 在支撐）；EMA20 向下 → 只做空（掛 Sell Limit 在阻力）。違反趨勢方向的掛單勝率低，不要掛。
3
確定掛單價格：Buy Limit = 支撐區上方 2–3 pips（避免剛好落在支撐被滑過）；Sell Limit = 阻力區下方 2–3 pips。
4
設定停損（SL）：Buy Limit 停損 = 支撐區下方 5–8 pips（跌破支撐 = 結構失效，離場）。Sell Limit 停損 = 阻力區上方 5–8 pips。停損距離通常 15–25 pips。
5
設定止盈（TP）：目標放在下一個明確阻力/支撐位（通常是前高或前低）。計算 RR = (TP 距離) ÷ (SL 距離) ≥ 2.0 才掛單，否則放棄此機會。
具體範例：支撐區 1.0838–1.0842，掛 Buy Limit @ 1.0845，SL @ 1.0820（-25 pips），目標前高 1.0895（+50 pips）→ RR = 50/25 = 2.0 ✓
方法二：EMA20 回測掛單（趨勢強勁時）
步驟操作具體說明
①確認強趨勢1小時圖 EMA20 持續傾斜，連續 5+ 根 K 在 EMA20 同側弱趨勢或橫盤不用此方法
②等待回調價格從高點回落靠近 EMA20回調 30–60% 之前波段最佳
③掛 Buy LimitEMA20 當前值 + 2–3 pips每根 K 棒 EMA 值不同，需每日重算
④停損EMA20 下方 12–18 pips若收破 EMA20 代表趨勢轉弱，停損出場
⑤止盈驗證 RR ≥ 2.0（前高距離 ÷ 停損距離）EMA 策略停損 15 pips → 需目標 30 pips
⑥有效期掛單設 4–8 小時有效EMA 值移動後原掛單位置意義消失
範例：EMA20 = 1.0848，掛 Buy Limit @ 1.0851，SL @ 1.0833（-18 pips），前高 1.0921（+70 pips）→ RR = 70/18 = 3.9 ✓✓
方法三：亞洲範圍突破掛單（倫敦開盤策略）
1
記錄亞洲時段範圍：每日 00:00–07:00 GMT，標記最高點（H）和最低點（L）。EURUSD 亞洲範圍通常 20–40 pips。
2
建立雙向掛單：Buy Stop @ H + 3 pips（突破上沿）；Sell Stop @ L - 3 pips（跌破下沿）。兩個同時掛，等待擇一觸發。
3
停損設定：Buy Stop 的 SL = L - 5 pips（若突破後又跌回範圍，停損）；Sell Stop 的 SL = H + 5 pips。停損距離 = 亞洲範圍 + 8–10 pips。
4
止盈確認 RR ≥ 2.0：目標 = 亞洲範圍幅度 × 2。若範圍 30 pips → 目標 60 pips。停損 = 範圍 + 8 = 38 pips → RR = 60/38 = 1.58（不足）。需要目標 = 38×2 = 76 pips 才達 RR 2.0。
5
一方觸發後立即取消另一方：Buy Stop 被觸發 → 立刻取消 Sell Stop，反之亦然。否則兩筆都觸發會互相損失。
6
有效期只到中午：設 Expiry = 12:00 GMT。超過倫敦時段的突破可信度低，未觸發自動取消。
注意：亞洲範圍策略的 RR 在範圍較小時（< 20 pips）更難達到 2.0，因為停損相對目標佔比過大。範圍 ≥ 25 pips 的日子更理想。
掛單前核查清單（每次必過）
核查項目要求說明
必要 趨勢方向1小時 EMA20 方向與掛單方向一致逆趨勢掛單勝率大幅下降
必要 支撐/阻力有效性同一區域有 ≥ 2 次觸及反應只有 1 次觸及的是「弱支撐」，不可靠
必要 RR ≥ 2.0 確認(目標 pips) ÷ (停損 pips) ≥ 2.0不足 2.0 就放棄，等下一個機會
強烈建議 數據日曆掛單有效期內無 Forex Factory 紅色事件數據前 30 分鐘取消所有掛單
強烈建議 有效期設定設 GTC 或指定時間，不讓掛單永久有效市場結構改變後舊掛單失去意義
建議 每日最多掛單數同時最多 2 筆未觸發掛單保持可管理，避免多單觸發同方向爆倉
MT4/MT5 下單步驟（Buy Limit 為例，Sell Limit 同理）
1
分析（下單前 5 分鐘）：確認趨勢 + 標記支撐 + 計算 RR。寫下：掛單價、SL 價格、TP 價格、手數。RR < 2.0 → 停止，不下單。
2
開下單視窗：MT4 → 右鍵圖表 → New Order（或按 F9）→ 展開 Type 下拉選單 → 選 Pending Order → 類型選 Buy Limit。
3
填寫五個欄位：
• Volume（手數）= 計算所得（例 0.50）
• Price（掛單價）= 支撐 + 3 pips（例 1.08430）
• Stop Loss = 掛單價 - 停損距離（例 1.08180）
• Take Profit = 掛單價 + 目標距離（例 1.08930）
• Expiry = 勾選並填入當日 21:00 GMT
4
複查三次後送出：① 方向是否是 Buy Limit（不是 Sell）→ ② SL/TP 數值是否正確（SL 在入場下方，TP 在入場上方）→ ③ 手數是否正確 → 確認 → Place。
5
確認掛單出現在 Terminal：MT4 下方 Terminal 視窗 → Orders 標籤 → 看到掛單行含有 SL 和 TP 欄位數值 → 確認正確。
6
觸發後確認：訂單觸發後自動移至 Trade 標籤 → 確認 SL 和 TP 已正確附加在持倉上（部分券商需在觸發後手動確認）。
7
每日結束前：掃描 Orders 標籤 → 取消所有超過有效期或市場結構已改變的未觸發掛單。
